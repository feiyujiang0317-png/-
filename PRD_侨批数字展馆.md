# PRD：侨批文化数字展馆 — 画册式交互体验

---

## 1. 产品概述

**一句话定位**：一款面向高校课堂投屏展示的**画册式交互数字展馆**，以12个精心设计的视觉画面串联侨批（海外华侨寄回家乡的信件与汇款）的百年历史脉络，通过克制的交互与沉浸的视觉质感，传递"纸短情长济天下"的文化情感内核。

---

## 2. 目标用户

| 角色 | 场景 | 需求 |
|------|------|------|
| 课堂观众（同学/老师） | 教室投影仪观看 | 被视觉吸引、被叙事打动、获得文化认知 |
| 展示者（团队成员） | 操作翻页/讲解 | 流畅的操作、清晰的叙事引导、可预期的展示时长 |
| 课程评分老师 | 评审作品 | 视觉美感 > 技术复杂度；文化传达 > 功能数量 |

> **核心洞察**：这不是一个"用户主动探索"的产品，而是一个"被观看的表演"。展示者操控节奏，观众沉浸接收。

---

## 3. 12个展区详细设计

### ① 封面页 — 情感定调

| 维度 | 描述 |
|------|------|
| **视觉定位** | 大面积的深色（墨色/旧纸色），中央一盏旧式油灯照亮画面，光影渐变柔和。左上角"纸短情长济天下"小字题签。 |
| **色调** | 暖棕底色 → 灯光暖黄，暗角浓重。RGB 参考：#2C1810 → #D4A574 |
| **核心文案** | 标题："侨批·纸短情长"；副标题："海外华侨家书与汇款的文化记忆"；底部小字："——交互式数字展馆——" |
| **交互方式** | 自动渐变淡入（约5秒完成，无需点击），随后显示"点击进入"提示符，点击后可进入② |
| **所需素材** | 旧式油灯剪影或手绘风格图片；纸张纹理背景图；书法字体标题字 |
| **情绪目标** | 引人入胜，安静肃穆，让观众沉下来 |

---

### ② 启封仪式 — 信封动效

| 维度 | 描述 |
|------|------|
| **视觉定位** | 画面中央是一个放置在木桌上的旧信封。信封正面朝观众，可见邮票和模糊的收件地址。点击后信封从封口处缓缓打开，内页信纸滑出，纸上内容漂浮浮现。 |
| **色调** | 桌面木色（深褐）+ 信封旧纸色（米黄）+ 内页文字（墨黑）。整体如旧照片质感。 |
| **核心文案** | 信封正面文字（年代久远模糊感）；内页浮现文字："见字如面"——四字浮现在信纸中央，随后淡出 |
| **交互方式** | 点击信封 → 拆封动画（约3秒）→ 信纸滑出 → "见字如面"浮现 → 自动过渡到③ |
| **所需素材** | 旧信封矢量/手绘设计；纸质纹理贴图；毛笔字"见字如面"图片 |
| **情绪目标** | 仪式感——仿佛亲自打开一封百年前的家书 |

### ③ 「難」字页 — 一字千钧

| 维度 | 描述 |
|------|------|
| **视觉定位** | 画面极简——全屏只有一个巨大的毛笔字"難"（1927年陈君瑞侨批原件），墨色淋漓，笔触清晰可见。字的边缘有轻微洇墨效果。背景是淡米色的旧纸纹理。点击后可浮现小字解说。 |
| **色调** | 纸白/旧纸色 + 浓墨黑为主。极简，无杂色。 |
| **核心文案** | 主体：一个字「難」（1927年陈君瑞写于马来亚）；浮现文案："一字千钧——一个'難'字，道尽华侨漂泊之苦"；底部小字："陈君瑞侨批，1927年" |
| **交互方式** | 该页静止 → 观众阅读 → 点击（或等待6秒自动）→ 字的旁边浮现批注文字 → 再点击进入④ |
| **所需素材** | 陈君瑞"難"字侨批高清晰图片或临摹矢量字；旧纸纹理背景 |
| **情绪目标** | 震撼——一个字的力量，胜过千言万语 |

### ④ 历史时间轴 — 侨批百年脉络

| 维度 | 描述 |
|------|------|
| **视觉定位** | 横向展开的长卷式时间轴。背景是淡米色宣纸纹理。时间轴以毛笔笔触画出的曲线贯穿画面，关键节点上有圆圆的小墨点标记。 |
| **色调** | 宣纸米白 + 墨色 + 朱砂红（关键年份标记） |
| **核心时间节点** | 1864年（有记载最早的侨批体系）→ 1927年（陈君瑞"難"字批）→ 1937-1945年（烽火侨批·抗战时期）→ 1957年（林朝光三千言长信）→ 1988年（侨汇体系收尾） |
| **交互方式** | 进入时自动从左侧开始缓缓滚动（约8秒走完全程）；也可手动点击节点查看简要说明；无节点显示时自动进入⑤ |
| **所需素材** | 时间轴设计稿；各节点简要说明文案；宣纸纹理背景 |
| **情绪目标** | 知性——建立历史框架感，让观众理解侨批不是零散事件，而是一个世纪的现象 |

### ⑤ 何为侨批 — 知识卡片

| 维度 | 描述 |
|------|------|
| **视觉定位** | 三张错落叠放的知识卡片，每张像一张打开的旧信纸。卡片纵向排列，之间有轻微阴影和错位。每张卡片上有图标+简洁文字。 |
| **色调** | 旧纸色卡面 + 墨色文字 + 淡红色图章式图标 |
| **核心文案** | 卡片1：**"侨"** = 海外漂泊的游子 / **"批"** = 闽南语"信"；卡片2：既是家书，又附汇款，一封侨批 = 一封信 + 一份侨汇；卡片3：跨越山海的情感纽带，1864-1988年持续124年 |
| **交互方式** | 页面进入时三张卡片依次从下方滑入（间隔0.5秒）；每张卡片可点击展开获得更详细说明（可选）；无操作则自动进入⑥ |
| **所需素材** | 三张卡片的设计稿；图标（信封/钱币/航海船等）；简要扩展说明文案 |
| **情绪目标** | 清晰——快速传达"侨批是什么"，为后续内容做知识铺垫 |

## 4. 视觉风格指南

### 4.1 整体风格定位

**关键词**：旧纸质感 · 墨色美学 · 暖光叙事 · 克制优雅

不是"数字化炫技"，而是"一本可以交互的老画册"。每一个画面都像一本精装画册的一个跨页。

### 4.2 色彩体系

| 色系 | 色值 | 用途 |
|------|------|------|
| 旧纸色（主色） | #F5E6C8 / #E8D5B0 | 背景底色，占画面60-70% |
| 墨色（主色） | #2C2C2C / #1A1A1A | 正文文字、图形轮廓 |
| 暖金（强调） | #C9A96E / #D4A574 | 标题文字、装饰线、重要数据 |
| 朱砂红（点缀） | #A52A2A / #8B0000 | 印章、关键标记、年份标注 |
| 深蓝（夜空） | #0A0E27 | 人物星河页背景 |
| 海蓝（南洋） | #1A2A3A / #2C4A5A | 下南洋页海洋 |

### 4.3 字体

| 层次 | 字体 | 用途 |
|------|------|------|
| 标题/书法字 | 毛笔手写体 / 行楷 | 封面标题、「難」字、章节标题 |
| 正文 | 宋体 / 楷体（思源宋体/Noto Serif SC） | 说明文字、故事内容 |
| 数据 | 衬线体或等宽体 | 数据数字 |
| 装饰文字 | 印章体/篆刻风格 | 批注、图章式说明 |

> **注意**：优先使用思源宋体（Noto Serif SC）和思源黑体（Noto Sans SC）作为web安全字体。书法标题字可预渲染为图片使用。

### 4.4 材质与纹理

- **纸张纹理**：所有展区背景应有轻微纸张肌理（噪点/纹理叠加），增强画册感
- **墨迹效果**：关键笔触（如「難」字、时间轴线条）做轻微的扩散/洇墨模拟
- **光线渐变**：封面/尾页使用径向渐变模拟油灯光照；其他页面使用柔和渐变增加层次
- **旧化处理**：照片/图片叠加轻微划痕、旧化效果，不追求高清还原而追求氛围

### 4.5 版式原则

- 每个展区均为 **满屏画面**（无浏览器UI干扰），类似PPT全屏模式
- 文字摆放遵循"气的流动"——留白充足，不在画面中塞满信息
- 重要文字使用"竖向排版"（如传统的竖排文字），增强年代感和仪式感
- 页面之间保持统一的页边距和视觉基线

### 4.6 转场设计

- 默认转场：**淡入淡出**（约1.2-1.5秒），柔和切换
- 特殊转场：④历史时间轴使用横向滑动（模拟长卷展开）
- **禁止**使用花哨的3D翻转、爆炸、旋转等动效——保持克制

## 5. 叙事节奏建议

| 展区 | 建议停留 | 情绪曲线 | 备注 |
|------|---------|---------|------|

## 6. 待确认问题

以下是在编写PRD过程中识别出的开放问题，请在开发前确认：

1. **「難」字侨批图片**：是否需要使用真实原件照片，还是由设计团队临摹手写？真实照片清晰度是否足够投屏展示？

2. **人物肖像使用权**：五个人物（陈君瑞、陈莲音、杨露义、林朝光等）是否有经过授权的历史照片？如果没有，是否改用象征性图形（信封、毛笔等）代替？

3. **数据精度确认**：侨汇数据（1864-1988年131.2亿美元、1939年1.56亿美元等）是否需要标注学术来源？是否应从论文中引用权威出处？

4. **⑨统一战线页的课程关联深度**：这一页需要与毛概教材的"第十章·文化自信"具体哪些理论概念直接挂钩？建议展示者（团队）明确"需要出现的理论关键词"。

5. **音频/背景音乐**：是否需要为展馆配背景音乐？如果需要，建议使用什么样的音乐风格（古琴/钢琴/环境音）？音频是否会与展示者讲解冲突？

6. **操作方式**：是使用键盘方向键翻页、鼠标点击、还是滚动滚轮？建议统一为 **点击+键盘左右键** 两种方式并行。

7. **投影比例**：教室投影仪的宽高比是16:9还是4:3？这影响设计画幅。建议按16:9设计，兼容性最好。

8. **⑩人物星河的5个人物**：抗战侨批部分（⑩第5个）定为"无名氏"还是使用已有素材中的具体人物（如洪铭通"四不写"、飞虎队华侨青年）？建议与指导老师确认。

9. **页面上是否需要页码/进度指示**：在底部显示"4/12"这类小标签，帮助观众建立空间感，还是不要破坏沉浸感？

10. **是否允许展示者"跳过"某些页面**：现场展示如时间紧张，是否提供快速导航功能？
